## 摘要
从包含多个连续动作的复杂时序描述中生成高质量视频是一个尚未解决的关键问题。现有方法受限于一种固有的权衡：按顺序向模型输入多个短提示词虽能提高动作保真度，但会损害时序一致性；而使用单个复杂提示词虽能保持一致性，却以牺牲提示词遵循能力为代价。我们将这一问题归因于两个主要原因：1) 视频内容与提示词之间的时序失配；2) 运动相关的视觉对象与其关联文本条件之间存在冲突的注意力耦合。为了应对这些挑战，我们提出了一种新颖的无需训练的注意力机制——时域可分离注意力（TS-Attn），它通过动态重排注意力分布，确保多事件场景下的时序感知和全局连贯性。TS-Attn 可以无缝集成到各种预训练的文本生成视频模型中，在 Wan2.1-T2V-14B 和 Wan2.2-T2V-A14B 上分别将 StoryEval-Bench 评分提升了 33.5% 和 16.4%，而推理时间仅增加 2%。它还支持在不同模型间即插即用地用于多事件图像生成视频任务。源代码和项目主页见 https://github.com/Hong-yu-Zhang/TS-Attn。

## Abstract
Generating high-quality videos from complex temporal descriptions that contain multiple sequential actions is a key unsolved problem. Existing methods are constrained by an inherent trade-off: using multiple short prompts fed sequentially into the model improves action fidelity but compromises temporal consistency, while a single complex prompt preserves consistency at the cost of prompt-following capability. We attribute this problem to two primary causes: 1) temporal misalignment between video content and the prompt, and 2) conflicting attention coupling between motion-related visual objects and their associated text conditions. To address these challenges, we propose a novel, training-free attention mechanism, Temporal-wise Separable Attention (TS-Attn), which dynamically rearranges attention distribution to ensure temporal awareness and global coherence in multi-event scenarios. TS-Attn can be seamlessly integrated into various pre-trained text-to-video models, boosting StoryEval-Bench scores by 33.5% and 16.4% on Wan2.1-T2V-14B and Wan2.2-T2V-A14B with only a 2% increase in inference time. It also supports plug-and-play usage across models for multi-event image-to-video generation. The source code and project page are available at https://github.com/Hong-yu-Zhang/TS-Attn.

#### 1. 论文的核心问题与整体含义（研究动机和背景）
论文针对的是**多事件视频生成（Multi-Event Video Generation）**中的关键瓶颈。当用户提供包含多个连续动作（如“猫先看球，然后拨弄球，最后叼走球”）的复杂描述时，现有模型面临“动作保真度”与“时序连贯性”的权衡：
*   **分段生成法**：将长提示词拆分为多个短提示词顺序生成，虽动作准确但视频片段间存在内容漂移和跳变，且推理耗时。
*   **单提示词生成法**：直接输入复杂长句，虽全局连贯但模型往往无法准确遵循所有指令，容易出现事件遗漏或时序幻觉。
**核心发现**：作者通过分析发现，失效的根本原因在于**时序失配**（视频帧未与对应动词对齐）和**注意力耦合冲突**（同一帧的注意力同时分散在多个不相关的事件词上）。

#### 2. 论文提出的方法论
作者提出了一种**无需训练（Training-free）**的注意力机制 —— **TS-Attn（Temporal-wise Separable Attention）**，旨在动态重构交叉注意力分布。

*   **核心思想**：解耦视频-文本注意力，使运动相关的视频 Token 在特定时间段内仅聚焦于对应的事件描述，并抑制其他无关事件的干扰。
*   **关键技术细节**：
    1.  **运动区域提取（Motion Region Extraction, MRE）**：通过提取主体（Subject）的交叉注意力图并进行自适应阈值处理，生成运动区域掩码 $M_s$，确保调制仅作用于执行动作的主体区域，避免背景闪烁。
    2.  **事件感知注意力调制（Event-aware Attention Modulation, EAM）**：
        *   **注意力重排（Attention Rearrangement）**：根据预设或 LLM 规划的时序区间，给对应事件的 Token 施加正向偏置（Bias），给非对应时段的事件 Token 施加负向偏置。
        *   **注意力增强（Attention Reinforcement）**：根据注意力分布的平滑程度，自适应地缩放偏置强度。如果分布太模糊，则加大增强力度。
*   **算法流程**：在扩散模型的早期去噪阶段（前 20%-40% 步），替换原有的交叉注意力计算公式，引入偏置项 $B$ 和增强因子 $R$：
    $$A = \text{softmax}\left(\frac{QK^\top + M_s \odot R(Q,K) \odot B(Q,K)}{\sqrt{d}}\right)$$

#### 6. 论文的主要结论与发现
*   **性能大幅提升**：在 Wan2.1-14B 上，TS-Attn 将 StoryEval-Bench 评分提升了 **33.5%**；在 CogVideoX-5B 上提升了 **57.3%**。
*   **时序控制有效性**：通过注意力重排，模型能精准地在指定时间点触发特定动作，解决了“事件遗漏”问题。
*   **即插即用**：该方法不改变原模型权重，可直接应用于任何基于交叉注意力的视频生成模型。
*   **时序分割的鲁棒性**：实验发现即使是简单的“均匀分割”也能获得不错的效果，说明 TS-Attn 对粗略的时序引导具有良好的容错性。

#### 7. 优点（亮点）
*   **高效免训练**：无需昂贵的微调过程，直接在推理侧解决复杂指令遵循问题。
*   **极低开销**：2% 的额外耗时几乎可以忽略不计，远优于现有的分段生成方案。
*   **通用性强**：支持 T2V/I2V，兼容多种主流开源架构，甚至能扩展到多主体生成和长视频生成（配合 LongCat）。
*   **物理合理性**：通过软性的注意力引导而非硬性掩码，保证了动作转换处的物理平滑性。

#### 8. 不足与局限
*   **依赖时序先验**：虽然均匀分割有效，但对于动作时长差异极大的复杂场景，仍需 LLM 或人工提供较准确的时间切分点。
*   **主体识别限制**：MRE 模块依赖于模型本身对主体词的理解，如果基础模型无法正确识别提示词中的主体，掩码提取可能会失效。
*   **视频长度限制**：实验主要集中在 5 秒左右的视频生成，虽然文中展示了长视频扩展潜力，但在超长视频中的累积误差尚未深入探讨。

## 摘要
从包含多个连续动作的复杂时序描述中生成高质量视频是一个尚未解决的关键问题。现有方法受限于一种固有的权衡：按顺序向模型输入多个短提示词可以提高动作保真度，但会损害时序一致性；而使用单个复杂提示词虽然能保持一致性，却以牺牲提示词遵循能力为代价。我们将这一问题归因于两个主要原因：1) 视频内容与提示词之间的时序错位；2) 运动相关的视觉对象与其关联文本条件之间存在冲突的注意力耦合。为了应对这些挑战，我们提出了一种新颖的、无需训练的注意力机制——时序可分离注意力（TS-Attn），它通过动态重新排列注意力分布，确保多事件场景下的时序感知和全局连贯性。TS-Attn 可以无缝集成到各种预训练的文本生成视频模型中，在 Wan2.1-T2V-14B 和 Wan2.2-T2V-A14B 上分别将 StoryEval-Bench 评分提升了 33.5% 和 16.4%，而推理时间仅增加 2%。它还支持在不同模型间即插即用地用于多事件图像生成视频。源代码和项目主页见 https://github.com/Hong-yu-Zhang/TS-Attn。

## Abstract
Generating high-quality videos from complex temporal descriptions that contain multiple sequential actions is a key unsolved problem. Existing methods are constrained by an inherent trade-off: using multiple short prompts fed sequentially into the model improves action fidelity but compromises temporal consistency, while a single complex prompt preserves consistency at the cost of prompt-following capability. We attribute this problem to two primary causes: 1) temporal misalignment between video content and the prompt, and 2) conflicting attention coupling between motion-related visual objects and their associated text conditions. To address these challenges, we propose a novel, training-free attention mechanism, Temporal-wise Separable Attention (TS-Attn), which dynamically rearranges attention distribution to ensure temporal awareness and global coherence in multi-event scenarios. TS-Attn can be seamlessly integrated into various pre-trained text-to-video models, boosting StoryEval-Bench scores by 33.5% and 16.4% on Wan2.1-T2V-14B and Wan2.2-T2V-A14B with only a 2% increase in inference time. It also supports plug-and-play usage across models for multi-event image-to-video generation. The source code and project page are available at https://github.com/Hong-yu-Zhang/TS-Attn.

#### 1. 核心问题与整体含义（研究动机和背景）
论文关注的是**多事件视频生成（Multi-Event Video Generation）**中的指令遵循难题。当用户提供包含多个连续动作（如“猫先看，再抓，最后跑”）的复杂提示词时，现有模型面临“动作准确性”与“时间连贯性”的权衡：
*   **分段生成法**：将长提示词拆分为多个短提示词顺序生成，虽动作准确但视频片段间存在跳变，且推理成本高。
*   **单次生成法**：直接输入长提示词，虽连贯性好但模型往往会忽略部分事件或产生时序幻觉（动作顺序错乱）。
**研究动机**：作者发现问题的根源在于交叉注意力层中视频 Token 与文本条件之间的**时序错位**和**注意力耦合冲突**。

#### 2. 论文提出的方法论
论文提出了一种名为 **TS-Attn（时序可分离注意力）** 的免训练（Training-free）插件机制，核心思想是在推理阶段动态重构交叉注意力分布。

*   **核心思想**：解耦视频 Token 与不同事件文本之间的关联，强制让特定时间段的视频帧主要关注对应的动作词，同时抑制无关事件的干扰。
*   **关键技术细节**：
    1.  **运动区域提取（Motion Region Extraction, MRE）**：通过提取主体（Subject）的语义布局图并进行阈值处理，生成运动区域掩码 $M_s$，确保注意力调制仅作用于运动主体而非背景。
    2.  **事件感知注意力调制（Event-aware Attention Modulation, EAM）**：
        *   **注意力重排（Attention Rearrangement）**：根据预设或 LLM 规划的时间段，为当前帧对应的事件 Token 施加正向偏置（Positive Bias），为非对应时段的事件施加负向偏置。
        *   **注意力强化（Attention Reinforcement）**：引入自适应强化因子，当注意力分布过于平滑（即模型犹豫不决）时，自动加大调制强度。
*   **算法流程**：在去噪的前 20% 阶段（运动形成的关键期），替换原始的 Cross-Attention，通过公式 $A = \text{softmax}(\frac{QK^T + M_s \odot R \odot B}{\sqrt{d}})$ 计算调制后的注意力图。

#### 3. 实验设计
*   **基础模型**：集成了 Wan2.1 (1.3B/14B), Wan2.2 (A14B), CogVideoX-5B 等主流 DiT 架构模型。
*   **任务场景**：涵盖文本生成视频（T2V）和图像生成视频（I2V）。
*   **Benchmark**：使用 **StoryEval-Bench**（包含 423 个提示词，分为人类、动物、创意等 7 个类别）。
*   **对比方法**：
    *   **开源模型**：Open-Sora, Vchitect-2.0, Pyramid-Flow, SkyReels-V2, MAGI-1。
    *   **多提示词方法**：MEVG, DiTCtrl。

*   **评估指标**：利用 GPT-4o 和 LLaVA-OV-Chat-72B 作为验证器，评估事件完整性、时序准确性和主体一致性。

#### 4. 资源与算力
*   **硬件**：所有实验均在 **NVIDIA A100 GPU** 上完成。
*   **训练时长**：该方法为**免训练（Training-free）**，无需任何微调或额外训练。
*   **推理开销**：由于仅在早期去噪步骤应用且不增加参数量，推理时间仅比原始模型增加约 **2%**（主要来自 LLM 辅助分割提示词的时间）。

#### 5. 实验数量与充分性
*   **实验规模**：在 StoryEval-Bench 的 423 个案例上进行了全量测试，并构建了 StoryEval-Bench-I2V 扩展集。
*   **消融实验**：详细验证了 EAM（注意力调制）、MRE（运动掩码）以及不同时序分割策略（均匀分割 vs LLM 规划 vs 人工标注）的影响。
*   **客观性**：通过多模型（Wan, CogVideoX）验证了通用性，并引入了第三方大模型（GPT-4o/LLaVA）进行双盲式量化评估，实验设计较为客观、公平。

#### 6. 主要结论与发现
*   **性能提升**：TS-Attn 在 Wan2.1-14B 上将 StoryEval-Bench 分数提升了 **33.5%**，在 CogVideoX 上提升了 **57.3%**。
*   **时序控制**：通过简单的注意力重排，可以有效解决多动作重叠或遗漏的问题。
*   **鲁棒性**：即使使用简单的“均匀时间分割”，该方法依然能取得显著提升，说明模型对精确的时间边界并不敏感。

#### 7. 优点
*   **免训练且即插即用**：无需昂贵的算力进行微调，可直接适配现有的 DiT 视频生成模型。
*   **极低开销**：在几乎不增加推理延迟的情况下，大幅提升了复杂指令的遵循能力。
*   **泛化性强**：同时适用于 T2V 和 I2V 任务，且支持多主体、多事件的复杂组合。
*   **物理一致性**：相比于分段拼接法，TS-Attn 保持了全局潜空间的一致性，避免了背景闪烁或主体突变。

#### 8. 不足与局限
*   **时序分割依赖**：虽然均匀分割有效，但对于动作时长差异极大的场景，仍需依赖外部 LLM（如 GPT-4o-mini）进行预规划。
*   **视频长度限制**：实验主要集中在 5 秒左右的视频生成，对于超长视频（如数分钟）的复杂叙事，可能仍需结合自回归生成技术。
*   **主体识别风险**：MRE 模块依赖于文本到区域的初始注意力映射，如果基础模型本身无法正确识别主体，TS-Attn 的效果会受限。

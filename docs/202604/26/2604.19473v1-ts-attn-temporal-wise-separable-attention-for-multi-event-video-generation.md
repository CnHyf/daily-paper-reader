## 摘要
从包含多个连续动作的复杂时间描述中生成高质量视频是一个尚未解决的关键问题。现有方法受限于一种固有的权衡：按顺序向模型输入多个短提示词虽能提高动作保真度，但会损害时间一致性；而使用单个复杂提示词虽能保持一致性，却以牺牲提示词遵循能力为代价。我们将这一问题归因于两个主要原因：1) 视频内容与提示词之间的时间错位；2) 运动相关的视觉对象与其关联文本条件之间存在冲突的注意力耦合。为了应对这些挑战，我们提出了一种新颖的、无需训练的注意力机制——时间维度可分离注意力（TS-Attn），它通过动态重排注意力分布，确保在多事件场景下的时间感知能力和全局连贯性。TS-Attn 可以无缝集成到各种预训练的文本生成视频模型中，在 Wan2.1-T2V-14B 和 Wan2.2-T2V-A14B 上，仅增加 2% 的推理时间，即可将 StoryEval-Bench 评分分别提升 33.5% 和 16.4%。它还支持跨模型的即插即用，用于多事件图像生成视频。源代码和项目主页见 https://github.com/Hong-yu-Zhang/TS-Attn。

## Abstract
Generating high-quality videos from complex temporal descriptions that contain multiple sequential actions is a key unsolved problem. Existing methods are constrained by an inherent trade-off: using multiple short prompts fed sequentially into the model improves action fidelity but compromises temporal consistency, while a single complex prompt preserves consistency at the cost of prompt-following capability. We attribute this problem to two primary causes: 1) temporal misalignment between video content and the prompt, and 2) conflicting attention coupling between motion-related visual objects and their associated text conditions. To address these challenges, we propose a novel, training-free attention mechanism, Temporal-wise Separable Attention (TS-Attn), which dynamically rearranges attention distribution to ensure temporal awareness and global coherence in multi-event scenarios. TS-Attn can be seamlessly integrated into various pre-trained text-to-video models, boosting StoryEval-Bench scores by 33.5% and 16.4% on Wan2.1-T2V-14B and Wan2.2-T2V-A14B with only a 2% increase in inference time. It also supports plug-and-play usage across models for multi-event image-to-video generation. The source code and project page are available at https://github.com/Hong-yu-Zhang/TS-Attn.

论文聚焦于**多事件视频生成（Multi-Event Video Generation）**中的关键瓶颈：如何让模型准确遵循包含多个连续动作的复杂提示词（如“猫先看，再跳，最后跑”）。
*   **现有痛点**：目前的模型在处理此类任务时面临“两难”：
    *   **分段生成法**：将长提示词拆分为多个短提示词顺序生成，虽动作准确但视频片段间存在明显的跳变和不连贯。
    *   **单次生成法**：直接输入长提示词，虽视频连贯但模型往往会忽略部分动作或发生时序混乱（时序幻觉）。
*   **核心发现**：作者通过分析注意力图发现，失效的根源在于**时序错位**（视频帧没对准对应的动词）和**注意力耦合冲突**（同一帧同时关注了多个不同时序的动作词）。

### 2. 论文提出的方法论：TS-Attn
作者提出了一种**无需训练（Training-free）**的注意力机制 **TS-Attn**，核心思想是动态重构交叉注意力分布，实现时序上的“解耦”与“对齐”。
*   **核心思想**：让视频中运动区域的 Token 在特定时间段内只关注对应的事件词，并抑制对无关事件词的关注。
*   **关键技术细节**：
    1.  **运动区域提取（MRE）**：利用主体（Subject）的语义布局生成掩码，识别出视频中真正执行动作的 Token 区域，避免误伤背景。
    2.  **事件感知注意力调制（EAM）**：
        *   **注意力重排（Rearrangement）**：根据预设或 LLM 规划的时序区间，给对应的事件词施加正向偏置（Bias），给非当前时段的事件词施加负向偏置。
        *   **注意力强化（Reinforcement）**：引入自适应强化因子，当注意力分布过于平滑（即模型“迷茫”时），自动加大调制力度。
*   **流程**：输入长提示词 -> LLM 自动切分时序区间 -> 在去噪早期阶段应用 TS-Attn -> 生成时序准确且连贯的视频。

### 6. 论文的主要结论与发现
*   **性能飞跃**：TS-Attn 在 Wan2.1-14B 上将 StoryEval-Bench 分数提升了 **33.5%**，在 CogVideoX 上提升了 **57.3%**。

### 7. 优点（亮点）
*   **免训练即插即用**：不需要昂贵的微调过程，直接适配现有的 DiT 架构视频模型。
*   **极低开销**：2% 的推理延迟增加在实际应用中几乎可以忽略不计。
*   **解释性强**：通过对交叉注意力机制的深入剖析找到病灶，方案直击痛点。
*   **多任务通用**：同时支持文本生成视频（T2V）和图像生成视频（I2V）。

### 8. 不足与局限
*   **时序切分的依赖**：虽然对精度要求不高，但仍需预先确定事件发生的先后顺序和大致时长（依赖 LLM 或人工输入）。
*   **极端复杂场景**：当视频中存在大量重叠动作或极微小物体运动时，基于语义布局的运动区域提取（MRE）可能会变得不够精确。
*   **模型架构限制**：目前主要针对基于 Cross-Attention 注入文本信息的 DiT 架构，对于其他注入方式（如纯自回归或非注意力机制）的适配性未讨论。

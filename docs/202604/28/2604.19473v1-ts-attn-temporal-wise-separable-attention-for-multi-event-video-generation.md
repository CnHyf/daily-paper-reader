## 摘要
从包含多个连续动作的复杂时序描述中生成高质量视频是一个尚未解决的关键问题。现有方法受限于一种固有的权衡：按顺序向模型输入多个短提示词可以提高动作保真度，但会损害时序一致性；而使用单个复杂提示词虽然能保持一致性，却以牺牲提示词遵循能力为代价。我们将这一问题归因于两个主要原因：1) 视频内容与提示词之间的时序错位；2) 运动相关的视觉对象与其关联文本条件之间存在冲突的注意力耦合。为了应对这些挑战，我们提出了一种新型的免训练注意力机制——时序可分离注意力（TS-Attn），它通过动态重新排列注意力分布，确保多事件场景下的时序感知和全局连贯性。TS-Attn 可以无缝集成到各种预训练的文本生成视频模型中，在仅增加 2% 推理时间的情况下，使 Wan2.1-T2V-14B 和 Wan2.2-T2V-A14B 在 StoryEval-Bench 上的得分分别提升了 33.5% 和 16.4%。它还支持在不同模型中即插即用，用于多事件图像生成视频。源代码和项目页面可在 https://github.com/Hong-yu-Zhang/TS-Attn 获取。

## Abstract
Generating high-quality videos from complex temporal descriptions that contain multiple sequential actions is a key unsolved problem. Existing methods are constrained by an inherent trade-off: using multiple short prompts fed sequentially into the model improves action fidelity but compromises temporal consistency, while a single complex prompt preserves consistency at the cost of prompt-following capability. We attribute this problem to two primary causes: 1) temporal misalignment between video content and the prompt, and 2) conflicting attention coupling between motion-related visual objects and their associated text conditions. To address these challenges, we propose a novel, training-free attention mechanism, Temporal-wise Separable Attention (TS-Attn), which dynamically rearranges attention distribution to ensure temporal awareness and global coherence in multi-event scenarios. TS-Attn can be seamlessly integrated into various pre-trained text-to-video models, boosting StoryEval-Bench scores by 33.5% and 16.4% on Wan2.1-T2V-14B and Wan2.2-T2V-A14B with only a 2% increase in inference time. It also supports plug-and-play usage across models for multi-event image-to-video generation. The source code and project page are available at https://github.com/Hong-yu-Zhang/TS-Attn.

*   **核心矛盾**：
    *   **多提示词流水线**：将复杂提示词拆分为多个短提示词分段生成，虽动作准确但视频段落间不连贯，且推理成本高。
    *   **单提示词直接生成**：保持了全局一致性，但模型难以精确遵循每个时序事件，导致提示词遵循能力（Prompt-following）下降。
*   **根本原因分析**：作者发现问题在于**时序错位**（视频内容与提示词时间点对不上）和**注意力耦合**（同一帧的视觉特征同时响应了多个不同时间的动作词）。

### 2. 论文提出的方法论：TS-Attn
TS-Attn 是一种**免训练（Training-free）**的插件式注意力机制，核心思想是在扩散模型的交叉注意力层中，动态地对不同时间段的视频 Token 进行“注意力重排”。

*   **核心流程**：
    1.  **时序分割**：利用轻量级 LLM（如 GPT-4o-mini）或均匀分割，将视频序列划分为对应不同事件的时间区间。
    2.  **运动区域提取 (MRE)**：通过分析交叉注意力图，识别出执行动作的主体（Subject）所在的视觉区域，生成运动掩码（Motion Mask），确保调制只作用于前景运动物体，不破坏背景。
    3.  **事件感知注意力调制 (EAM)**：
        *   **注意力重排 (Rearrangement)**：在特定时间区间内，增强视频 Token 对当前对应事件词的注意力，同时抑制对其他时间段事件词的注意力。
        *   **注意力强化 (Reinforcement)**：根据注意力分布的平滑程度，自适应地缩放权重，确保动作特征被有效注入。
*   **公式逻辑**：在原始注意力分数上增加一个偏置项 $B$ 和强化因子 $R$，仅在运动掩码 $M_s$ 覆盖的区域生效。

### 5. 实验数量与充分性
*   **实验规模**：在 4 个主流开源模型架构上进行了验证，涵盖了 T2V 和 I2V 两种任务。

*   **客观性**：对比了多种闭源 SOTA 模型，并使用了第三方视觉大模型（LLaVA）进行双重验证，实验设计较为全面、客观。

### 6. 主要结论与发现
*   **性能提升**：在 Wan2.1-14B 上，StoryEval-Bench 分数提升了 **33.5%**；在 CogVideoX-5B 上提升了 **57.3%**。
*   **效率优势**：在保持单次推理一致性的前提下，其提示词遵循能力超过了需要多次推理的 DiTCtrl 等方法。
*   **鲁棒性**：即使时序分割不够精确（如简单的均匀分割），TS-Attn 依然能显著提升多事件生成的质量。

### 7. 优点与亮点
*   **免训练与即插即用**：无需昂贵的微调，可直接适配任何基于 DiT 架构的视频生成模型。
*   **高效率**：几乎不增加推理延迟，解决了长视频生成中多提示词拼接导致的计算冗余。
*   **解耦控制**：通过 MRE 模块实现了前景动作与背景一致性的有效解耦，避免了背景闪烁。
*   **通用性**：不仅支持单主体多事件，还扩展到了多主体、场景级变换及交互式长视频生成。

### 8. 不足与局限
*   **依赖外部规划**：虽然支持均匀分割，但最佳效果仍依赖于 LLM 对提示词的时序切分，增加了系统复杂性。
*   **基础模型限制**：TS-Attn 无法突破基础模型本身的最大生成时长或物理规律限制（如模型本身不会画某个动作，该方法也无法凭空创造）。
*   **掩码精度**：运动区域提取依赖于初始的交叉注意力质量，如果基础模型对主体的定位极度不准，MRE 的效果可能会受限。
